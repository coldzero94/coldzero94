## GitHub Skyline

To add your GitHub Skyline:
1. Visit https://skyline.github.com/coldzero94/2025
2. Take a screenshot or download the 3D model image
3. Save it as `github-skyline.png` in this directory

Or use the GitHub CLI:
```bash
gh skyline 2025
# Then move the generated image here
```
